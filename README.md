# StatsTellStories-Volume1

Welcome to our repo! This is where we collect all the notebooks, scripts, and materials for the projects we showcase on our YouTube channel **"StatsTellStories"**.

## What's in here?

Each video has its own dedicated folder with the corresponding code, notebooks, and (if possible) small datasets. That way you can jump straight to a video and check out the full code, or try it out yourself.

This list grows with every new video, so check back from time to time!

## Feedback & questions

Got a question about a specific video, or a suggestion for a new topic? Feel free to leave a comment under the corresponding YouTube video — we check regularly.

Have fun exploring and rebuilding! 🚀
